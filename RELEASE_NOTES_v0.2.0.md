# v0.2.0

## New
- `--json` output for machine-readable audit results
- `--strict` mode for CI-friendly warning enforcement
- Additional checks for template boilerplate and effectively empty markdown files
- Improved summary and exit-code behavior

## Why it matters
This release makes `openclaw-workspace-doctor` more useful both for humans and automation. You can now use it interactively or wire it into CI and scheduled checks.
