# Contributing

Thanks for contributing to `openclaw-workspace-doctor`.

## Development

```bash
python3 -m pip install -e .
openclaw-workspace-doctor /path/to/workspace
```

## Pull requests

Please keep PRs small and focused:
- bug fix
- new audit check
- documentation improvement
- test coverage

## Reporting issues

When filing an issue, include:
- OpenClaw version
- OS / architecture
- sample workspace layout
- expected vs actual output
